# Creating a new Project

Make sure you have npm, node.js and create-react-app tool.

```
Create-react-app project-name
```

OR

```
git clone https://github.com/shouryasrivastava/discord-clone
```

## Starting a project

```
npm start
```

## Project Strucutre

```
src
├── components
│   ├── App.js
│   ├── SignIn.js
│   └── index.js
├── index.css
└── index.js
```
